# ClipGo

ClipGo is a short video sharing platform where users can watch, upload, and interact with short videos.

## Features

- User Signup and Login
- Short Video Feed
- Upload Videos
- Video Player
- Search / Explore Content
- Creator Profiles
- Notifications
- Messages (Chat)
- Followers / Following System
- Settings Page
- About Page
- Privacy Policy & Terms
